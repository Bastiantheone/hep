hepmc
=====

``hepmc`` is a pure ``Go`` implementation of the ``C++`` ``HepMC``
library.

## Installation

```sh
$ go get github.com/go-hep/hepmc
```

## Documentation

Doc is on [godoc](http://godoc.org/github.com/go-hep/hepmc)

## Example

```go
package main

import (
  "github.com/go-hep/hepmc"
)

func main() {
  
}
```
